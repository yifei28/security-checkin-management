# 都豪鼎盛内部系统 - 用户使用说明书

## 系统概述

都豪鼎盛内部系统是一个专为安保管理设计的Web应用程序，用于管理安保人员、工作地点和签到记录。系统支持两种用户角色：普通管理员和超级管理员。

## 系统访问

### 登录系统
1. 打开浏览器访问系统网址
2. 在登录页面输入您的用户名和密码
3. 点击"登录"按钮
4. 登录成功后会自动跳转到管理后台

### 账户要求
- 用户名：最少3个字符
- 密码：最少6个字符

## 主要功能模块

### 1. 控制台（首页）
- **功能**：显示系统概览和重要统计信息
- **访问**：登录后的默认页面
- **内容**：
  - 系统整体数据概览
  - 最新签到记录
  - 关键性能指标

### 2. 员工管理
- **功能**：管理安保人员信息
- **操作指南**：
  - **查看员工列表**：页面显示所有安保人员的基本信息
  - **添加新员工**：
    1. 点击"添加员工"按钮
    2. 填写必要信息（姓名、工号、联系方式等）
    3. 分配工作地点
    4. 保存信息
  - **编辑员工信息**：
    1. 在员工列表中找到目标员工
    2. 点击"编辑"按钮
    3. 修改相关信息
    4. 保存更改
  - **删除员工**：
    1. 选择要删除的员工
    2. 点击"删除"按钮
    3. 确认删除操作

### 3. 单位管理
- **功能**：管理工作地点和站点信息
- **操作指南**：
  - **查看单位列表**：显示所有工作地点的详细信息
  - **添加新单位**：
    1. 点击"添加单位"按钮
    2. 填写单位名称、地址、坐标等信息
    3. 设置签到有效范围
    4. 保存单位信息
  - **编辑单位信息**：
    1. 在单位列表中选择要修改的单位
    2. 点击"编辑"按钮
    3. 更新相关信息
    4. 保存更改
  - **删除单位**：
    1. 选择要删除的单位
    2. 点击"删除"按钮
    3. 确认删除（注意：删除前请确保没有员工分配到该单位）

### 4. 签到记录
- **功能**：查看和管理员工签到数据
- **操作指南**：
  - **查看签到记录**：
    - 按日期筛选记录
    - 按员工筛选记录
    - 按单位筛选记录
    - 查看签到状态（成功/失败）
  - **筛选和搜索**：
    1. 使用日期选择器选择时间范围
    2. 选择特定员工或单位
    3. 查看筛选结果
  - **导出数据**：
    1. 设置筛选条件
    2. 点击"导出"按钮
    3. 选择导出格式
    4. 下载文件

## 用户角色说明

### 普通管理员
- 可以查看所有模块的信息
- 可以管理员工和单位信息
- 可以查看签到记录
- **限制**：无法管理其他管理员账户

### 超级管理员
- 拥有普通管理员的所有权限
- 额外权限：
  - **系统管理员管理**：
    - 添加新的管理员账户
    - 删除现有管理员账户
    - 设置管理员权限级别

## 系统管理员功能（仅超级管理员）

### 访问管理员管理页面
1. 以超级管理员身份登录
2. 导航到"系统管理员"页面
3. 查看当前所有管理员账户

### 添加新管理员
1. 在管理员管理页面点击"添加新管理员"
2. 填写以下信息：
   - 用户名（唯一）
   - 密码
   - 选择角色类型（普通管理员/超级管理员）
3. 点击"添加管理员"完成创建

### 删除管理员
1. 在管理员列表中找到要删除的账户
2. 点击"删除"按钮
3. 确认删除操作

## 操作建议

### 数据安全
- 定期更换密码
- 不要与他人共享账户信息
- 退出系统时请点击"退出登录"

### 最佳实践
1. **员工管理**：
   - 及时更新员工信息变更
   - 确保员工工号的唯一性
   - 及时删除离职员工记录

2. **单位管理**：
   - 准确设置地理坐标
   - 合理设置签到有效范围
   - 定期检查单位信息的准确性

3. **签到记录**：
   - 定期查看异常签到记录
   - 及时处理签到失败的情况
   - 定期备份重要数据

## 常见问题解答

### Q: 忘记密码怎么办？
A: 请联系超级管理员重置您的密码。

### Q: 为什么签到失败？
A: 可能的原因：
- 员工不在指定的签到范围内
- 网络连接问题
- 人脸识别失败

### Q: 如何批量导入员工信息？
A: 目前系统暂不支持批量导入，需要逐个添加员工信息。

### Q: 系统支持哪些浏览器？
A: 推荐使用最新版本的Chrome、Firefox、Safari或Edge浏览器。

### Q: 数据多久备份一次？
A: 系统数据会自动备份，具体备份策略请咨询系统管理员。
